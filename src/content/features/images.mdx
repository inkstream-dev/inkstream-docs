# Images

## Usage

```tsx
import { availablePlugins } from "@inkstream/starter-kit"

<RichTextEditor
  plugins={[availablePlugins.image]}
  toolbarLayout={['image']}
/>
```

## Features

- Drag-and-drop upload
- Paste from clipboard
- Resizable via drag handles
- Markdown export: `![alt](url)`
