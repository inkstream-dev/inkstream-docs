# Lists

## Plugins

| Plugin | Export |
|--------|--------|
| Bullet list | `bulletListPlugin` |
| Ordered list | `orderedListPlugin` |
| Indent | `indentPlugin` |

## Usage

```tsx
import { availablePlugins } from "@inkstream/starter-kit"

<RichTextEditor
  plugins={[availablePlugins.lists, availablePlugins.indent]}
  toolbarLayout={['lists', '|', 'indent', 'outdent']}
/>
```

## Input rules

| Type | Trigger |
|------|---------|
| Bullet list | `- ` or `* ` at line start |
| Ordered list | `1. ` at line start |

## Nesting

Use `Tab` to indent a list item and `Shift-Tab` to outdent.
